Git

Mis on Git?

Git lokaalne vahend, mida kasutatakse andmete (koodi) haldamiseks. Git jaotab need andmed repositooriumi sisse.

Mis on repositoorium?

Repositoorium on koht, kuhu andmete ajalugu on salvestatud.

Mis on "Branch"?

"Branch" on repositooriumi sees olev koht, mis hoiustab kasutaja commit'e.

Mis on Commit?

Commit tähistab branchis oleva koodi erinevaid etappe.

Kuidas oma projekti Gitiga siduda?

Kui Git on alla laetud, siis tuleb avada giti terminal ja navigeerida oma projekti kausta ning käskude abil git projektiga siduda.

Git'i käsud: 

    git init - initsialiseerib repositooriumi 

    git add . - lisab projekti failid giti süsteemi

    git commit -m - salvestame projekti mingis etapis, kus anname commitile nime, kuhu kirjutatakse muudatuste kohta info. 

    git log - viimaste commitide vaatamiseks 

    git reset --hard (commit id) - lisame commit id, mis lubab kustutada selle commiti ja kõik pärast seda.

    git checkout --. - liigub viimase commit'i juurde. 

    git checkout -b (branch name) - moodusta uue branch'i, milles on ka kõik eelmise branch'i commitid.  

    git merge (name of branch) - seob kaks branchi kokku. 

    git branch -D (name of branch) - kustutab branch'i.
