Github

Mis on Github?

Github veebipõhine lahendus, mis hoiustab Git'i repositooriume.

Kuidas Githubi kasutada?

Githubi kasutamiseks tuleb teha Githubi kasutaja ning seejärel luua repositoorium.

Repositoorimisse informatsiooni või kood lisamiseks kasutame git'i, et meie olemasolev repositoorium githubi lisada.

Selle jaoks: 

        Kasutame git'i käsku git remote add origin (githubi repositooriumi veebileht)

        Remote add märgib seda, et tahame kohaliku ja githubi repositooriumi ühendada.

        git push -u origin master - lükkab branch'i githubi repositooriumisse. Git push üle võib mõelda ka kui "update" või "publish".

Kuidas jagada oma repositooriumit oma kolleegidega?

Githubi repositooriumis on "clone or download" valik, kust saab oma projekti githubi linki näha.

See link tuleks kasutusele panna terminalis käsuga git clone (github link) ning terve see repositooriumi sisu kopeeriti "clone" folderisse.

Kuidas githubi repositooriumist andmeid kätte saada?

git fetch - võtab repositooriumist andmeid, kuid ei ühenda neid sinu omaga

git pull - võtab andmed ning ühendab need olemasolevate andmetega.
